# Specification

## Summary
**Goal:** Create a whitewater kayak design showcase website where users can browse kayak designs and admins can add new entries.

**Planned changes:**
- Build backend data model to store kayak designs (name, manufacturer, year, image URL, external link)
- Implement backend methods to add and retrieve kayak designs sorted by year
- Create frontend gallery displaying kayak cards with images, details, and external links
- Add admin form to submit new kayak designs
- Apply water-themed visual design (teals, aqua, blues, whites) with energetic typography
- Display hero background and logo from generated assets

**User-visible outcome:** Users can browse a visually appealing gallery of whitewater kayak designs sorted by year. Admins can add new kayak entries through a form. The site features a dynamic water-themed design with a rapids hero background and kayak logo.
